# Medical Chatbot

## Overview

The Medical Chatbot is a web-based application that provides basic
healthcare information through a conversational interface. Users can ask
questions about symptoms, diseases, and general health topics, and the
chatbot responds with helpful medical guidance.

This project demonstrates how Artificial Intelligence and Natural
Language Processing can be used to build healthcare support systems that
provide quick and accessible medical information.

## Features

-   Interactive chatbot interface
-   Provides information about symptoms and diseases
-   Instant responses to health-related queries
-   Simple and user-friendly web interface

## Technologies Used

-   Python
-   Flask
-   HTML

### 3. Run the Application

### 4. Open in Browser
